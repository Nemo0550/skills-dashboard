# My Skills Dashboard

A personal dashboard of every Claude Cowork skill and slash command I have installed — searchable, filterable, and mobile-friendly.

Built as a single self-contained HTML file. Just open `index.html` in any browser, or view it live (deployed on Vercel).

## Categories
Skills are grouped into: System & Tools, Automation, Documents, Research, Communication, Productivity, and Design.

## Updating
The skill list lives in the `SKILLS_DATA` array inside `index.html`. To add a skill, add a new entry with its name, command, description, and category.
